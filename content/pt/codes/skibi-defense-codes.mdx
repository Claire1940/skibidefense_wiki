<Section icon={Target} title="Processo de Resgate Passo a Passo">

O resgate de códigos nesta experiência difere um pouco dos jogos padrão do Roblox que apresentam um botão dedicado na interface principal do usuário. Em vez disso, este jogo utiliza o console de chat do jogo para processar comandos promocionais. Este sistema requer uma inserção precisa para garantir que o servidor reconheça o comando e conceda os itens à sua conta.

<Steps>
  <Step num="1" title="Inicie o Jogo">
    Abra a plataforma Roblox e inicie o cliente oficial do jogo. Certifique-se de estar conectado a um servidor estável para evitar desconexões de rede durante o processo de resgate.
  </Step>
  <Step num="2" title="Abra a Caixa de Chat">
    Clique no ícone de chat localizado no canto superior esquerdo da tela ou pressione a tecla "/" no seu teclado para focar automaticamente no campo de entrada do chat.
  </Step>
  <Step num="3" title="Insira o Comando do Código">
    Digite o prefixo do comando seguido pelo código. O formato deve ser exato. Digite "/redeem" seguido de um espaço e o nome específico do código.
  </Step>
  <Step num="4" title="Confirme e Envie">
    Pressione a tecla Enter para enviar a mensagem. Uma mensagem de confirmação aparecerá no registro do chat detalhando as recompensas creditadas ao seu inventário.
  </Step>
</Steps>

Se você encontrar problemas ao tentar reivindicar suas recompensas, consulte a tabela de solução de problemas abaixo para identificar e resolver erros comuns de digitação.

| Problema | Causa Comum | Solução Recomendada |
| :--- | :--- | :--- |
| Mensagem de Código Inválido | Falta do prefixo de barra ou capitalização incorreta | Copie e cole o código diretamente no chat |
| Mensagem de Código Expirado | O código atingiu a data de validade | Verifique os canais da comunidade para obter códigos ativos atualizados |
| Tempo Limite de Rede | Alta latência do servidor ou conexão de internet instável | Entre em um servidor diferente e tente o comando novamente |
